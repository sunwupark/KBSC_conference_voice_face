## Function:
  1. 사람 등록하기
  2. 분석하기
  3. 부가적인 것들
  
  앱 사용법: 
  1. 사람 등록하기 
  - 이름을 TextField안에 작성한다
  - 얼굴에 초점을 맞춘다
  - 버튼을 누른다
  
  2. 분석하기
  - 얼굴에 초점을 맞추는 상태에서 중지버튼을 누를 때까지 대화한다
  
  3. 부가적인 것들
  - Navigation View, Bottom Navigation View 등을 추가하여 편하게 바로 이동 가능하다
  - ? 버튼을 누르면 간단한 설명이 나온다
